Version 1.10

  • Add the ability to export a notebook document to HTML export, using IPython's nbconvert feature.
  • If you quit the application with a notebook document open, the application will try to reopen that document at the next launch.
  • Add a “Clear All Output” menu command to clear the output of all cells in a notebook document.

Version 1.9

  • Fix build problem that made system-wide Python modules, such as Foundation, unavailable.

Version 1.8

  • Add a new application icon, designed by Jono Hunt.
  • Add PIL, needed by scikit-image
  • Extend server startup timeout, it was too short, especially on slower systems.
  • Properly embed the MathJax JavaScript library so opening notebooks without an Internet connection doesn’t show a warning message.

Version 1.7

  • Add the scikit-image module.
  • Add the bokeh module.

Version 1.6

Note that this version requires OS X 10.9 / Mavericks. Do not install this version if you still need support for 10.8.

  • Update to the 1.1.0 release of IPython Notebook to bring it up to the 1.0 series.
  • Add the scikit-learn module.
  • Various small user interface improvements.
  • Added a new OpenCV example, courtesy of Jan Erik Solem.
  • Add support for full-screen mode.
  • Added ability to customize the notebook's CSS in the preferences, and added a default CSS that improves the appearance of Python code in cells by using the Menlo font.
  • Add ⌘-R shortcut to execute current cell.

Version 1.4

  • You can now add directories, such as your custom virtualenvs, to the application’s PYTHONPATH in the preferences. This lets you add modules in addition to the ones that come with the app or OS X.

Version 1.3

  • First public release
